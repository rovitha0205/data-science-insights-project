#  Integrated Data Science Project

## Title
Integrated Analysis of Student Performance and Retail Sales using Data Science Techniques

---

## Objective
This project demonstrates the application of data science techniques such as:
- Data Cleaning
- Data Filtering
- Correlation Analysis
- Regression
- Visualization

---

## Datasets Used

### 1. Student Dataset
Contains subject marks:
- Maths
- Science
- English

### 2. Retail Dataset
Contains:
- Product_ID
- Quantity
- Unit Price
- Date

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

##  Key Concepts Implemented

### 1. Data Cleaning
Missing values were handled and duplicates removed.

### 2. Correlation Analysis
Heatmap used to identify relationships between subjects.

### 3. Regression Analysis
Linear Regression used to predict Science marks based on Maths.

### 4. Data Visualization
- Heatmap
- Bar charts
- Line charts

### 5. Business Analysis
Retail data used to identify:
- Total revenue
- Top-selling products
- Monthly trends

---

## Output Explanation

### ✔ Student Analysis
- High correlation between Maths and Science
- Top students identified using average scores

### ✔ Regression
- R² score indicates relationship strength between subjects

### ✔ Retail Insights
- Few products contribute most revenue
- Sales trend varies monthly

---

##  Final Insights
- Students good in Maths tend to perform well in Science
- Sales are not uniform across months
- Revenue distribution is uneven

---

## Conclusion
This project successfully demonstrates how data science techniques can be applied to extract meaningful insights from datasets.
